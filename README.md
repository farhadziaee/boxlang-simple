## BoxLang simple web application

### Run
```sh
docker compose up --build
```

### Nginx
* Check this file `./nginx.conf`

### WebSocket
* Check this files
  * `./WebSocket.cfc`
  * `./public/script.js`

### MySql
* extension: `bx-mysql`
* Queries: `./classes/db.bx`

### Web Requests & Routing
* `./classes/core/WebRequest.bx`
* `./classes/core/WebRoute.bx`
* `./app/router.bx`
